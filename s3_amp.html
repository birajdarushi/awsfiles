AWS Cloud Project

AWS S3 + Amplify + WAF Deployment Demo

A comprehensive showcase of modern cloud deployment using AWS services for secure, scalable static website hosting.

2nd August 2025
2 weeks development
Cloud Architecture

Project Overview

Complete deployment pipeline using AWS cloud services

Objective

Demonstrate how to quickly deploy and secure a static website using AWS S3 and AWS Amplify Hosting. The goal was to launch a site rapidly with minimal configuration while leveraging modern cloud best practices.

Approach

Used AWS S3 for static file hosting and AWS Amplify for one-click deployment & CI/CD. AWS WAF and CloudFront CDN were activated automatically for security and global distribution.

Result

Successfully deployed a secure, globally distributed website with automatic HTTPS, CDN acceleration, and web application firewall protection - all with beginner-friendly setup.

Technology Stack

AWS services and tools used in this project

Amazon S3

Static website hosting with high availability and durability

AWS Amplify

One-click deployment and continuous integration/deployment

CloudFront CDN

Global content delivery with automatic HTTPS and caching

AWS WAF

Web application firewall for protection against common threats

GitHub Integration

Optional automated pipelines for advanced deployment workflows

Key Highlights

What makes this deployment special

01

Seamless Deployment

Amplify made deployment and firewall setup seamless — all integrated with just a few clicks, perfect for rapid prototyping and production deployments.

02

Automated CI/CD Pipeline

Option to pull code directly from GitHub for advanced, automated pipelines. Kept simple for this demo but scalable for complex workflows.

03

Global Distribution

CloudFront CDN globally distributes content for blazing fast access and automatic HTTPS, ensuring optimal performance worldwide.

04

Built-in Security

Protection from web threats with AWS WAF without manual firewall rules, providing enterprise-level security out of the box.

05

Beginner-Friendly

Less deep-diving or manual AWS configuration — perfect for quick launches and learning cloud deployment fundamentals.
